El elemento HTML <canvas> (lienzo), se usa para dibujar gráficas, al vuelo, vía JavaScript. Canvas es solo un contenedor para gráficas. Debes usar JavaScript para realmente dibujar sobre el “lienzo”. Canvas representa solo un área rectangular en la página web, por defecto no tiene bordes ni contenido. Canvas tiene varios métodos para el dibujo de trazos, cajas, círculos, texto y adición de imágenes. En esta página se dibujan varios ejemplos utilizando Canvas y JavaScript.
