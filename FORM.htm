La etiqueta <form> define un formulario diseñado para recolectar entradas del usuario. Un formulario contiene diferentes elementos de entrada, como campos de texto, casillas de verificación, botones radiales, listas y otros. La etiqueta <input> define una entrada o campo y es el elemento más importante dentro de un formulario. La forma como se muestra cada campo depende del atributo type y del navegador usado. Normalmente los datos recogidos en el formulario son enviados a un servidor o al programa de correos. Con CSS es posible darle un aspecto moderno y agradable a los formularios.
