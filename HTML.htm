HTML es la sigla en inglés de HyperText Markup Language (lenguaje de marcas de hipertexto). Es el elemento de construcción más básico de una página web y se usa para crear y representar visualmente una página web. Determina el contenido de la página web, su estructura, pero no su funcionalidad. Otras tecnologías distintas de HTML son usadas generalmente para describir la apariencia de una página web (CSS) o su funcionalidad (JavaScript). En esta página se muestran algunas etiquetas HTML.
