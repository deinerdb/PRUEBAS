Uso de Etiquetas

Cuando comencemos a escribir una etiqueta, siempre debemos usar "<Etiqueta>" y terminar con "</Etiqueta>". Dentro de ellas escribimos la etiqueta que nos guste.
Ejemplo:
-La etiqueta "Title" sirve para colocarle un título a nuestra página.
